Exercise 3: Accessing Global

What will the following code log to the console and why? Don't run the code until you have tried to answer.

Answer

This code will log the String 'This is local' because the variable myVar referenced within the function is the same as the one in the global scope (no new variable is declared within the function). When the function is called, it reassigns the variable to a new value, so that when it is logged on line 9, it is the new value which is logged.

Discussion

The someFunction function changes the value of myVar when it gets reassigned a new value. This is because JavaScript variables at the global scope can be accessed and changed inside an inner scope.
