Exercise 6: Pass by Value

What will the following code log to the console and why? Don't run the code until you have tried to answer.

Answer

This code will log the Number 7 to the console. The variable a is passed in to the the function myValue as an argument. This variable acts as a pointer to the Number 7. Within the function definition, the parameter b acts as a pointer to the the value of the argument that is passed in (i.e. the Number 7). Within the function body, the parameter, which is a local variable within the scope of the function, is reassigned to the return value of the operation to sum the Number to which b is pointing (i.e. 7) and the Number 10, which is the other operand. So b is assigned to 17 but a remains unchanged and so is still assigned to 7 (which is what is logged).

Discussion

JavaScript is "pass-by-value". Passing variable a to myValue binds the local variable b to its value of 7 and this doesn't interfere with the outer variable a. Thus, when the program logs the value of a to the console its value remains the same.
