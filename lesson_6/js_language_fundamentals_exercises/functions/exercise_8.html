Exercise 8: Passing Objects

What will the following code log to the console and why? Don't run the code until you have tried to answer.

Answer

This code will log the Array [1, 2, 10] to the console. The variable a is declared in the global scope and assigned to the array object [1, 2, 3]. This variable acts as a pointer to the array; when a is passed in as an argument ot the function, the function parameter b is assigned to the array. Since objects are mutable, the operation to set the value at index 2 of the array mutates the array itself. It is the mutated array that is logged on line 8.

Discussion

Recall that JavaScript is pass-by-value. In this exercise, the tricky part is understanding the "value" passed. For JavaScript, when passing "objects" (i.e arrays) the value is the reference to it. As such, when the program executes the statement b[2] += 7 on line 4 it actually executes it on the object pointed at by a in the outer scope since it a got a reference to it on line 7. Consequently, when it logs the value of a on line 8 it logs [1, 2, 10].
