Exercise 10: Convert a Signed Number to a String

In the previous exercise, you reimplemented a function that converts non-negative numbers to strings. In this exercise, you're going to extend that function by adding the ability to represent negative numbers.

You may not use any of the standard conversion functions available in JavaScript, such as String(), Number.prototype.toString() or doing ' ' + number. You may, however, use integerToString from the previous exercise.
