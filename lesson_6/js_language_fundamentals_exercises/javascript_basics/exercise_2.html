Exercise 2: Truthy Falsey

Go over the program. What does it log to the console at lines 7, 11, 15, and 19? Is it what you expected? Why or why not?

Answer

Line 7 logs 'Hello', since myBoolean evaluates as true. Line 11 doesn't log 'World', since myString is truthy but is negated by the bang in the condition. Line 15 does log 'World' since the empty array myArray is truthy, and the double bang converts this to the boolean true. Line 19 logs '!' since, although myOtherString is falsy, the right hand side of the 'or' condition, myArray is truthy

Discussion

Recall that it is only the following values that are falsey:

  • false
  • null
  • undefined
  • 0
  • NaN
  • ''
If it isn't any of those, consider the value to be truthy.
