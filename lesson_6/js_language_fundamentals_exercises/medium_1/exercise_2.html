Exercise 2: Conditional Loop

Review the code below. The program should log numbers that are multiples of 3 from 0 to 9. Will the expected behavior happen? Why or why not?

Answer

NO, because the first time the while loop hits a number for which number % 3 equals 0, the else branch of the conditional will not be executed, and so the value of i will not be incrmented. It will result in an infinite loop. in fact this will happen from the first loop since 0 % 3 === 0 is true.

Discussion

It becomes an infinite loop because the value of i is never incremented. The expression (i % 3 === 0) will always be true and as such the else clause that contains the incrementer never gets executed. When using a while loop, you have to be wary about managing the variables involved with the iteration and the stopping condition. In the case of this exercise, i should have been incremented at every iteration.
