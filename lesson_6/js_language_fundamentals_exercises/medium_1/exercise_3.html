Exercise 3: Multiplication Table

The following program is expected to log a multiplication table for the numbers 1 up to 10 to the console. Read through the code. Will the given code produce the expected result? Why or why not?

Answer

No, because the outer for loop only increments up until 9, (the condition is while i is ;es than 10, so i never actually reaches 10.)

Discussion

This is an example of an off-by-one bug. The value of i, which is responsible for the rows of the table, only goes up to 9. As such, the entire row for the values of 10 x 1, 10 x 2...10 x 10 is not logged.
