Exercise 8: Product of Sums

The following program returns the product of the totals of two arrays of numbers. Review the code. Is it producing the expected behavior? Why or why not?

Answer

This won't work for two reasons: 1) because the total() function doesn't assign an iniital value to the sum variable, and so each iteration is just adding the number for that iteration to undefined which results in sum being NaN; 2) even if you fix that, sum is not explicitly returned, so productOfSums() will be multiplying undefined with undefined, which results in NaN.

Discussion

The current implementation returns NaN. There are two things that are problematic with this code. The first is that the total function does not have an explicit return value, therefore it returns undefined. Since it is undefined the product of total(array1) * total(array2) becomes undefined * undefined, which evaluates to NaN. The second issue is that the sum variable in the total function is not assigned with a value and is therefore undefined.
