Exercise 5: What's My Value?

What will the following program log to the console? Can you explain why?

Answer

Line 4 will log 3. 3.4 is not a valid index, so this will be added as a property of the Array obejct rather than 'Oranges' being added as an array item. Since only array items contribute to the length of an array, the length remains 3.

Line 5 will log 4. Even though the 3.4: 'Oranges' pair doe not contribute to the array length, 3.4 is a key of the array object and so is counted along with the array indices by Object.keys

Line 8 will log 3, for the same reasons given for line 4(-2) is not a valid index so will not count towards the array length.

Line 9 will log 5, for the same reasons given for line 5 (-2 counts as an object key).

Discussion

Recall that Arrays are implemented with Objects internally in JavaScript. One thing that differentiates the two is that arrays have a length property, but objects don't. In a way, you can see arrays as special objects with only integer values (from 0 up to 2 32 - 1) as keys and with an extra length property that tracks how many such key/value pairs exist in the object.
