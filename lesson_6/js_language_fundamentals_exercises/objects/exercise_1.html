Exercise 1: Literal

Read the code and identify the bug in it. Think about it first before running it.

Answer

The bug is that the a in myObject[a]; is not in quote marks so JavaScript thinks its a variable and not a property of the myObject object.

Discussion

myObject[a] triggers a ReferenceError. In order to access the value of a key using square bracket notation ([someKey]), the key has to be a string value. When the value is numeric it gets converted to a string using the toString() method. If it's a variable, it looks up the value (converting it to a string if necessary) and uses the value to get the corresponding property value for it. The statement myObject[a] triggered a ReferenceError because it tried to get the value of a variable that isn't declared.
