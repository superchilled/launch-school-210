Exercise 5: Array Object 1

Read the code and say what values it will log to the console.

Answer

Line 2 will log 'a', since this is the value at index 0 of myArray. Line 3 will log undefined, since using a negative index returns undefined unless there is a matching property in the array (which is an object). Line 8 will log 'd', since -1 was added to the object as a property on line 4, with a value of 'd'. Line 9 will log 5, since this is the value of the object property 'e' added on line 5; since arrays are objects we can add properties to them, but these properties are not array items. Line 10 will log ['a', 'b', 'c', 'f', '-1': 'd', , 'e': 5] since this will show all the array items, and then the properties the properties (index 3 was assigned to 'f' on line 6).
