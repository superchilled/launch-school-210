Exercise 6: Array Object 2

The function below takes an array as argument and returns its average. The user of the function put in the statements below and was expecting the function to return 5 to test it. What is the value returned?

Answer

The value returned will be 10, since the for loop references the -2 and -1 properties as well as the array items, it sums 5 4 times, making a total of 20, which it then divides by array.length, which is 2, producing 10.

Discussion

A non-positive integer key is not considered as part of an array's length. As such the length is only 2 when the sum of the elements were divided (20 / 2).
