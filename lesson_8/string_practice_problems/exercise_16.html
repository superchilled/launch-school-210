Launch School 210 | String and Text Processing: Practice Problems - 16

Write a program that asks for a user's name, then greets the user with that name. If the user appends a ! to his name (e.g., Bill!), the computer should "yell" its greeting: that is, it should log everything to the console in uppercase. You can check whether the name ends with a ! with String.prototype.endsWith(). You can remove the ! from the user's name with String.prototype.slice().
