Exercise 1: Odd Numbers

Log all odd numbers from 1 to 99, inclusive, to the console. Log all numbers on separate lines.

Answer

There are a number of ways to approach this problem. Part of the problem focuses on iterating through numbers starting at 1 and ending at 99; this part can be solved with a simple for loop within which the value of i can be incremented by 2, since every other number will be odd.

Another option would be to increment through every number from 1 to 100 and use a conditional to check if it is odd and then log it if it is. This solution might be slightly les abstract, but would also be less efficient.
